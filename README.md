## 📖 Описание
GUI-приложение на Python (`tkinter`) для генерации случайных цитат из встроенной базы. Поддерживает фильтрацию по автору и теме, сохраняет историю генераций в JSON, позволяет добавлять новые цитаты с валидацией ввода. Проект разработан в учебных целях для демонстрации работы с GUI, файловой системой, модулем `random` и контроля версий Git.

## 🛠 Технологии
- Python 3.8+
- `tkinter` (встроенная библиотека)
- `random`, `json`, `os`
- Git / GitHub
